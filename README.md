# 红柚起始页 (Red Pomelo Start Page)

一个极简、美观且功能强大的浏览器起始页，采用现代化的毛玻璃（Glassmorphism）设计风格，集成了搜索、书签管理、专注模式、音乐播放等实用功能。

## 主要特性

### 界面与交互
- **极简设计**：采用 Apple 风格的 UI 设计，细腻的毛玻璃效果和流畅的动画。
- **深色模式**：内置日/夜间模式切换，带有精美的 SVG 切换动画。
- **响应式布局**：完美适配桌面端和移动端设备。

### 搜索与导航
- **多引擎搜索**：支持 Google、Bing、百度、DuckDuckGo 等主流搜索引擎。
- **快速链接**：可自定义的常用网站快捷方式网格。
- **书签管理**：侧边栏书签管理器，支持从浏览器（如 Edge）导入书签。
- **桌面书签**：支持在主页直接显示书签文件夹内容。

### 实用工具
- **专注空间 (Focus Mode)**：
  - 内置番茄钟计时器（25/15/45/60分钟预设）。
  - 待办事项列表（Todo List）。
  - 专注数据统计（今日专注时长、番茄数）。
  - 沉浸式全屏体验。
- **天气组件**：实时天气显示，点击可查看详细天气数据（风速、湿度、体感温度、紫外线等）。
- **每日一言**：每天更新一句励志名言。

### 媒体播放
- **网易云音乐集成**：
  - 支持通过歌单 ID 加载网易云音乐歌单。
  - **悬浮播放器**：页面左下角常驻悬浮窗，支持切歌和播放控制。
  - 侧边栏完整播放列表。
- **环境音效**：内置多种白噪音（雨声、森林、咖啡馆等），助你专注或放松。

### 个性化定制
- **背景设置**：
  - 预设多种纯色和渐变背景。
  - **自定义上传**：支持上传图片或视频作为背景（使用 IndexedDB 本地存储，无需服务器）。
  - 网络图片/视频链接支持。
- **布局调整**：支持调整背景显示方式（填充、适应等）。

## 快速开始

本项目为纯静态网页，无需复杂的后端部署。

1. **下载代码**：
   ```bash
   git clone https://github.com/rojim666/index.git
   ```
2. **运行**：
   直接在浏览器中打开 `index.html` 文件即可使用。

## 项目结构

```
.
├── index.html      # 页面主体结构
├── styles.css      # 样式文件 (CSS3, Variables)
├── script.js       # 业务逻辑 (Vanilla JS, IndexedDB)
└── _logo.svg       # 网站图标
```

## 使用指南

- **切换主题**：点击左上角的太阳/月亮图标。
- **打开设置**：点击右下角的悬浮按钮（+号）打开功能面板。
- **导入书签**：打开收藏夹面板 -> 点击导入按钮 -> 选择浏览器的导出的 HTML 书签文件。
- **加载歌单**：打开音效面板 -> 切换到"网易云音乐" -> 输入歌单 ID -> 点击加载。
